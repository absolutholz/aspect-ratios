    functions

    create-ratio

    @function create-ratio($dimensions, $description: "", $key: "") { ... }

    Description

    Create a ratio map. Ideally not one already defined in the $aspect-ratios variable

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $dimensions

    A list of x and y dimensions (ex: 1 1 or 16 9)

    		List— none
    $description

    A brief description of the ratio

    		String""
    $key

    The key for the generated map (ex: "1x1" or "16x9"). If an empty or no string is passed, one will be generated from the dimensions in the format "[x]x[y]".

    		String""

    Returns

    Map —

    ($key: ("description": $description, "dimensions": $dimensions))

    aspect-ratio-box

    @mixin aspect-ratio-box($x, $y, $pseudo: false) { ... }

    Description

    Create a box element with a fixed aspect ratio via the css padding-hack.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $x

    The x (horizontal) value of the desired ratio

    		Number— none
    $y

    The y (vertical) value of the desired ratio

    		Number— none
    $pseudo

    Should the box be generated via a pseudo element. Set this to true when flex-basis alone defines the parent width.

    		Boolfalse

    Content

    This mixin allows extra content to be passed (through the @content directive).

    Output

    Padding-top value to create the desired aspect ratio, either nested directly under the parent selector or, optionally, as a before pseudo element.
